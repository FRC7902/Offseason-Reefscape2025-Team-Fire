# StrafeRightCommand.txt - strafe right slowly

## requirements:
- move robot right without rotating
- run while button held

## pseudocode:

StrafeRightCommand extends Command:

constructor:
  require swerve subsystem

initialize():
  // nothing

execute():
  call swerve.strafe with:
    - direction: 1 (right)
    - speed multiplier: 0.05 (slow)

end():
  // nothing (swerve default command takes over)

isFinished():
  return false
  // runs while button held

## notes:
- slow speed for precise alignment
- positive direction = right
- mirrors StrafeLeftCommand
